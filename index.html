프로세스 / 안내

현실적인 작업 흐름으로, 깔끔한 진행

불필요한 커뮤니케이션을 줄이기 위해, 작업 범위를 명확히 안내드립니다.

작업 프로세스

작업 진행 과정 타임라인

  1. 작업 전 문의

    결제 전 문의로 작업 가능 여부와 일정/범위를 확인합니다.

  2. 결제 및 작업 확정

    결제 완료 시 작업이 확정됩니다.

    작업은 입금순으로 진행됩니다.

  3. 자료 전달

    텍스트 / 이미지 등 필요한 자료를 전달 받습니다.

    자료 전달이 늦어질 경우 작업에 딜레이가 있을 수 있습니다.

  4. 제작 진행

    확정된 구성/자료 기준으로 제작합니다.

    진행 중 큰 방향 변경은 어렵습니다.

  5. 컨펌·수정

    패키지에 포함된 수정 횟수 내에서 컨펌을 진행합니다.

  6. 최종 전달

    수정 반영 후 최종본 전달.

    최종본 전달 이후에는 수정이 불가능합니다.
